Interaction Design II

Interaction Design II covers topics including: basics of coding, coding principle, logic theories, visual design and interaction, 3D modeling and basisc of game design.

Interactive Visuals

Project Description

Students learned the basics of programming using Javascript and the P5.js Library.

The students assignment was to create a simple interactive experiences using HTML, Javascript and P5.js. Student were encourages to incorporated images and audio.

Sophisticated 3D Models

Project Description

Students learned how to use the basics of 3D modeling tool Blender. Students were taught how to create and manipulate basic polygon models.

In addition, students were taught how to create and apply digitial materials to their models to give them characters. Students also learn how to create and position digital lights and cameras to capture their world.

The students project was to plan and build a sophisticated 3D model that incorporated materials, lights and cameras.

Build a Virtual World

Project Description

Students learn the basics of using the Unity game engine. Students learn how to create and place models, audio and images into Unity.

In addition, students learn how to program basic #C in order to make objects move, collide and interact with one another.

Final Interactive Projects

Project Description

Students created their own interactive projects using code, visual graphics presented them as an online experience.
