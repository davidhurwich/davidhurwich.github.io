[About Vector Project]

This was the second project we did for the class. It was done using Illustrator. The objective was to create a simple character and said character over a raster background.

Going into this project, I decided to continue the theme of my Raster Project. This theme being Snapchat. I referenced two characters from the anime SSSS.Gridman and placed them over a stock raster image I found online. I then put a transparent black text box with text over it to emulate a "Snap".
