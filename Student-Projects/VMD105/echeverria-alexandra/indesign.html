The fourth project that we had done in class was with Adobe InDesign, which focused on Print Layout!

This project seemed to be the most challenging out of all of the projects for me. For some reason, it was hard for me to wrap my head around some of InDesign’s ins and outs. I did however have a fun time working on the postcards that you see above.

One of the most challenging parts for me was to make sure that everything looked good together. Not only that, but to make sure I was doing everything correctly in terms of size, resolution, etc. I ended up messing up on the proper resolution and color spaces. I think next time if I were to re-do this project I would take more time making sure that I understand everything.
