DIGM15: Web Design Fundamentals

Introduction to the basic design elements required to create a visually compelling website or user experience. Emphasis on design and layout, vector based graphics, and style consistency.

Students produced work showing web skills as well as principles of basic user experience

Choose Your Own Adventures Prototypes Small Business Websites

Choose Your Own Adventure

Project Description

Students designed simple interactive choose your own adventure stories.

Students created pages of the story using images, HTML and CSS. Students linked the stories to create imersive interactive experiences.

App Prototypes

Project Description

Students created smart phone application prototypes using Figma. Students conducted research to learn about the needs and likes of potential users. Users considered elements of information architecture, interactive theory and use of microinteractions.

Small Business Website

Project Description

Students worked in small groups to create a website for a local business. The students began by researching the needs of the stakeholder (owner) of the business, as well as the potentials customers. Student map out competitors to the business, the potential target customers as well as possible visual styles including wireframes and mockups.

Students then built the website based on the mockup using HTML and CSS. Students then loaded the website to a server to become a working website.
